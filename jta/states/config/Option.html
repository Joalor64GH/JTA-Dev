class Option

package jta.states.config

@:directlyUsed

Represents a configuration option in the settings menu.

Constructor

new(name:String, type:OptionType, value:Dynamic)

Creates an option with the specified name, type, and value.

Parameters:

name

The name of the option.

type

The type of the option.

value

The initial value of the option.

Variables

name:String

The name of the option displayed in the settings menu.

@:value(false)showPercentage:Bool = false

Whether or not to display a '%' sign for Integer and Decimal types.

type:OptionType

The type of the option.

value:Dynamic

The current value of the option.

onChange:Dynamic ‑> Void

A function to be called when the option value changes.

Methods

@:value({ direction : 0 })changeValue(direction:Int = 0):Void

Changes the value of the option based on the direction input. Adjusts the value by a step amount.

Parameters:

direction

The direction to adjust the value.

execute():Void

Executes the function associated with this option if it is of type Function. Attempts to call the function stored in the value property and logs an error if unsuccessful.

toString():String

Converts the option to a string for display purposes.

Returns:

A string representation of the option, formatted according to its type.
